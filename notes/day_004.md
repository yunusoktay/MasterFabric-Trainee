# Day 004

## Focus

- Review variables and data types
- Review control flow
- Review null safety
- Solve FizzBuzz
- Build a simple number guessing game

## Run

From the repository root:

```bash
dart run dart/day_004.dart
```

Use a fixed target number for easier testing:

```bash
dart run dart/day_004.dart 42
```

## Outcome

Day 4 is about practice and review. The goal is to reuse the ideas from the first three days in small programs.

FizzBuzz is good practice for loops and conditions. It checks if a number is divisible by `3`, `5`, or both.

The guessing game uses:

- `Random()` to create a target number
- `stdin.readLineSync()` to get user input
- `int.tryParse()` to safely convert text into a number
- `while` and `if/else` to repeat the game and guide the user

This day also reinforces null safety with a short review example that uses `?.` and `??`.
