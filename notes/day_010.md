# Day 010

## Focus

- Understand what collections are
- Work with `List` (create, add, remove, access, iterate)
- Work with `Map` (create, add, remove, access, iterate)
- Iterate with `for-in` and `forEach`

## Run

From the repository root:

```bash
dart run dart/day_010.dart
```

## Outcome

Day 10 introduces Dart's two most common collection types.

### List

An **ordered** collection of items accessed by index (starting at 0).

| Operation | Code |
|---|---|
| Create | `final list = ['a', 'b', 'c'];` |
| Add | `list.add('d');` |
| Remove | `list.remove('b');` |
| Access | `list[0]` |
| Length | `list.length` |

### Map

A collection of **key-value pairs**. Keys are unique.

| Operation | Code |
|---|---|
| Create | `final map = {'Alice': 30};` |
| Add / Update | `map['Bob'] = 25;` |
| Remove | `map.remove('Alice');` |
| Access | `map['Alice']` |
| All keys | `map.keys` |
| All values | `map.values` |

## Iteration

```dart
for (final item in list) { ... }      // for-in
list.forEach((item) => print(item));  // forEach

for (final entry in map.entries) { ... }
map.forEach((key, value) => ...);
```

## When to Use Which

- Order matters, items can repeat → **List**
- Lookup by unique identifier → **Map**
