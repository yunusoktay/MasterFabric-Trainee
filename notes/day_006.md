# Day 006

## Focus

- Define a class in Dart
- Create objects from a class
- Add properties and methods
- Use constructors to initialize objects

## Run

From the repository root:

```bash
dart run dart/day_006.dart
```

## Outcome

Day 6 introduces custom types with classes.

The example uses:

- `Student` to show a basic class
- Properties like `name`, `age`, and `favoriteTopic`
- Methods like `introduce()`, `study()`, and `completeLesson()`
- A constructor to create each object with starting values

Key ideas:

- A `class` is a blueprint
- An object is an instance created from that blueprint
- Properties store data that belongs to the object
- Methods define behavior that belongs to the object
- A constructor sets up the object when it is created
