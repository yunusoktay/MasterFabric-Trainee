# Day 005

## Focus

- Build a command-line calculator
- Practice user input with `stdin.readLineSync()`
- Use `switch` for operator selection
- Handle errors: invalid input, division by zero

## Run

From the repository root:

```bash
dart run dart/day_005.dart
```

## Outcome

The calculator asks the user for two numbers and an operator, then prints the result.

Key concepts used:

- `stdout.write()` to prompt without a newline
- `double.tryParse()` to safely convert text to a number
- `switch` to pick the correct arithmetic operation
- Early return with `null` to handle errors without crashing
